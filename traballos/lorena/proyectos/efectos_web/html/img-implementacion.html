Imágenes de

sustitución

Reemplazar una imagen del sitio web cuando el usuario realiza una acción determinada aporta dinamismo y funcionalidad a la página.

Para realizar el efecto de sustitución de imágenes contamos con diferentes técnicas según el lenguaje de programación.

Utilizando únicamente HTML  con el atributo onmouseover  podremos ver que la imagén de la derecha cambia cuando el ratón pasa por encima.

Este mismo efecto se puede conseguir utilizando CSS  mediante la pseudoclase :hover,  y con JavaScript  además contamos con muchas otras posibilidades a la hora de aplicar efectos en las transiciones.

Imágenes

secuenciales

Son un conjunto de imágenes que se muestran en un orden específico para crear la ilusión de movimiento o para ilustrar un proceso paso a paso.

Para realizar este efecto lo mejor es utilizar la librería JQuery  de JavaScript.

Este efecto se puede relaizar con cualquier objeto de la página web adaptando el código.

Plugins y

librerías

Existen multitud de plugins  que se pueden añadir a un sitio web y que ofrecen elementos funcionales y con distintos efectos que pueden ser de gran utilidad. Los más comunes son galerías de imágenes o sliders, ya que son muy dinámicos y vistosos.

Según el lenguaje utilizado, habrá que enlazar una o más bibliotecas en el documento HTML. El documento CSS  servirá para modificar el diseño del plugin (color, tamaño), y los archivos JavaScript  permiten cambiar parámetros como el tiempo entre transiciones o tipos de efectos
